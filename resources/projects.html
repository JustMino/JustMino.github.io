Coin Flip Prediction Version 1 This is a simple python script that runs to predict a coinflip outcome. The prediction and the coinflip itself are both random and this was set up to be used as a standard.

Coin Flip Prediction Version 2 This is the 2nd Version of the Coinflip Prediction. This version uses a counting system where it would predict either side while having a chance more on a certain side.

Coin Flip Prediction Version 3 This is the 3rd Version of the Coinflip Prediction. This version uses keeps count of heads and tails. It will check the ratio of heads/tails and if it goes over a certain number it will predict either side.
